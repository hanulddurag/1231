3-1. 추론하며 읽기 | 동네 쓰레기를 하루아침에...

1

추론(推論)의 정의

이미 알고 있는 하나 또는 둘 이상의 판단을 근거로 삼아 다른 판단을 이끌어 내거나, 글에 드러나지 않은 글쓴이의 의도나 관점을 짐작하는 정신적 활동임.

2

추론하며 읽기의 구체적 방법

•

책의 표지 그림, 차례, 글의 제목, 소제목 등을 사전에 훑어보며 전체적인 내용을 미리 짐작함.

•

글에 직접 드러난 정보(시각 자료, 핵심 단어, 문맥, 접속어 등)를 적극적으로 분석함.

•

독자 개인이 기존에 머릿속에 축적하고 있는 배경지식(스키마)과 과거의 직간접적 경험을 글의 내용과 입체적으로 연결함.

•

생략되거나 행간에 숨겨진 글쓴이의 숨은 의도, 가치관 및 관점을 논리적으로 미루어 짐작함.

3

추론하며 읽기의 긍정적 효과

•

수동적인 태도에서 벗어나 지식과 의미를 독자 스스로 재구성하는 능동적인 독자로 성장함.

•

글을 읽는 과정에서 인지적인 몰입도가 증대되어 본문의 세부 흐름에 한층 더 깊이 집중할 수 있음.

•

글의 심층적이고 입체적인 내용을 한결 정밀하게 이해할 수 있어 기억이 장기 기억으로 전이됨.

•

넛지 효과를 활용해 다양한 사회 문제를 성공적으로 해결한 실제 사례들을 흥미로운 시각 자료와 대조하여 제시함.

•

사례마다 글쓴이의 핵심 의도와 관점이 명징하게 드러나도록 감각적인 소제목을 부여함.

•

기존의 통제 및 규제 중심 해결책과 넛지 효과를 활용한 자율적 해결책의 진행 과정 및 결과를 선명하게 대비함.

Q8. 앞부분의 내용을 고려할 때, "여러분이라면 천 마디의 잔소리와 부드러운 메시지 중 어느 쪽에 마음이 움직일까요?"라는 질문에 담긴 의도는 무엇일까?

글에 제시된 두 가지 해결 방법 비교

기존의 해결 방법

  • 경고문, 과태료, 시시티브이(CCTV) 단속 등
  • 일방적이고 강압적인 지시와 통제
  • 사람들의 마음을 움직이기 어려움 (반발심 유발)
  • 천편일률적이며 근본적인 문제 해결이 힘듦 (단기적 효과)

넛지 효과를 활용한 해결 방법

  • 미니 화단, 피아노 계단, 지도 화장지, 도로 흰색 가로선 등
  • 부드러운 개입으로 사람들의 흥미와 공감 유발
  • 스스로 올바른 선택을 하도록 유도 (자율성, 자발적)
  • 문제에 대한 장기적이고 창의적인 근본적 해결 가능

글쓴이의 관점과 의도 (추론의 핵심 요약)

관점

강압적인 법적 규제나 천편일률적인 잔소리 대신, 사람들의 심리를 부드럽게 이끌어 자발적이고 능동적인 참여를 유도하는 넛지 효과를 대단히 유용하고 긍정적인 사회 치유의 수단으로 깊이 존중하고 신뢰함.

의도

사회의 고질적이고 복잡한 고난들을 슬기롭고 우아하게 타개해 나가기 위해서는 단순 일방적인 통제나 경고 보단, 대중이 스스로 변동할 기회를 창출해내는 따뜻하고 독창적인 발상(아이디어)의 창안이 시급하게 요구됨을 일러주기 위함.

아이들은 어른보다 추위를 덜 탈까?

제재 기본 정리

갈래

설명하는 글(설명문)임.

성격

해설적, 과학적, 분석적임.

소재

몸속 갈색 지방(아이들이 추위를 덜 타는 이유)임.

주제

아이들이 어른보다 추위를 덜 타는 과학적 원리(까닭)임.

특징
•

독자의 흥미와 공감을 유발함.

•

질문과 답변의 형식을 사용하여 과학적 원리를 독자에게 알기 쉽게 설명함.

•

기능이 다른 백색 지방과 갈색 지방의 특성을 대조하여 핵심 개념을 부각함.

글의 구성 및 날개 질문

처음 (처음 ~ 124:12)

일상적 경험을 통한 의문 제기

어릴 적 '다방구' 놀이 일화를 소개하며 "정말 아이들은 어른보다 추위를 덜 탈까?"라는 호기심을 이끌어 냄.

Q1. 글제목과 '다방구' 일화를 볼 때, 글쓴이는 이 글에서 어떠한 내용을 다루려는 것일까요?

중간 (124:13 ~ 125:21)

1. 백색 지방과 갈색 지방

몸속 '백색 지방'과 '갈색 지방'의 역할을 대조하고, 미토콘드리아가 많은 갈색 지방이 백색 지방을 태워 열을 발생시키는 원리를 설명함.

백색 지방
  • 쓰고 남은 열량을 몸에 저장함.
  • 우리 몸에 있는 지방의 대부분을 차지함.
갈색 지방
  • 다량의 미토콘드리아를 포함하여 갈색을 띰.
  • 열을 많이 만들어 백색 지방을 태우는 역할을 함.
  • 갓난아이들은 지방 중 약 5%를 차지하여 체온 유지에 필수적임.

Q2. 갓난아이들 몸속의 '갈색 지방'은 어떠한 역할을 한다고 추론할 수 있을까요?

2. 성장과 갈색 지방의 감소

나이가 들면서 근육이 발달하면 몸을 떨어 스스로 열을 낼 수 있게 되어 몸 안의 갈색 지방이 점차 줄어들게 됨을 설명함.

Q3. 본문 및 시각 자료를 바탕으로, 아이들이 성장하면서 우리 몸속의 '갈색 지방'이 줄어드는 까닭을 추론해 볼까요?

끝 (125:22 ~ 끝)

아이들이 추위를 덜 타는 이유에 대한 결론

갈색 지방의 비율 차이로 인해 결과적으로 아이들이 어른보다 추위를 덜 탄다는 사실을 요약하며 글을 마무리함.

Q4. 결론적으로, 아이들이 어른에 비해 상대적으로 추위를 덜 타는 까닭은 무엇인가요?

function saveResultToSheet(data) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // 데이터가 아예 없는 시트일 때 상단 헤더행 자동 생성
  if (sheet.getLastRow() === 0) {
    var headers = ["학년", "반", "번호", "이름", "총점"];
    for(var i = 1; i <= 10; i++) {
      headers.push("Q " + i + " 응답", "Q " + i + " 정오");
    }
    sheet.appendRow(headers);
  }
  
  var row = [
    data.grade,
    data.class,
    data.number,
    data.name,
    data.score
  ];
  
  // Q1~Q10 응답과 정오(OX)를 순서대로 배치
  for(var i = 1; i <= 10; i++) {
    row.push(data["q" + i + "_ans"], data["q" + i + "_eval"]);
  }
  
  sheet.appendRow(row);
  return "success";
}
